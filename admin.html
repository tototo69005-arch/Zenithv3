Guide d'utilisation

Comment utiliser le panneau d'administration Zenith.

Circuits sur mesure

Cliquez sur Modifier pour ouvrir le formulaire d'un circuit. Vous pouvez changer : nom (FR/EN/MG), description, prix, durée, images (URL Unsplash ou votre propre hébergement), programme jour par jour, inclus/exclus. Cliquez Nouveau circuit pour en créer un. Supprimer est irréversible (jusqu'au prochain reset).

Voyages organisés

Gérez les voyages à dates fixes. Le champ Places prises met à jour automatiquement la barre de progression et l'alerte rouge (≤ 3 places restantes). Quand un voyage est complet, mettez "places prises = places max" — le bouton Réserver affiche "Complet".

Photos

Utilisez des URLs d'images directes. Pour Unsplash : https://images.unsplash.com/photo-XXXX?w=700&q=80. Pour vos propres photos : hébergez-les sur imgbb.com (gratuit) et copiez l'URL directe.

Sauvegarde

Toutes les modifications sont sauvegardées dans le localStorage du navigateur — elles persistent sur votre appareil. Pour partager les modifications sur un autre appareil ou les sauvegarder définitivement, exportez les données (bouton ci-dessous) et partagez le fichier JSON à intégrer dans data.js.

Zone dangereuse

Réinitialise toutes les données aux valeurs d'origine (data.js). Irréversible.
